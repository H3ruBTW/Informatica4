DESCRIZIONE ESERCIZI

  1. modificare il colore e lo sfondo dei pulsanti presenti nella pagina al passaggio del mouse
  2. Convertire il testo in maiuscolo durante la digitazione nel campo codice_fiscale
  3. Cambiare lo sfondo della casella di testo (tag input) quando questa prende il focus e ricambiarla quando lo perde (eventi focus e blur).
  4. Nascondere un testo dopo il click impostando nell’elemento la proprietà hidden = true;
  5. Mostrare un messaggio di errore quando il campo ripetipassword contiene un valore diverso dal campo password
  6. nel campo password cambiare il colore dello sfondo in base alla rispetto dei requisiti impostati per la password.
